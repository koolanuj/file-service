Usage:

This application provides following REST APIs to fetch file attributes. Sample calls (in local environment):
1) To get attributes of a file:
http://localhost:8080/fileservice/filedetail?fileName=C:/Program Files/file-name

2) To get attributes of ALL files in a directory recursively:
http://localhost:8080/fileservice/dirdetail?dirName=C:/Program%20Files
